# Shop-Reward-System-Tech
Shop Performance Reward System

The Shop Performance Reward System is a web-based application developed using Python Flask and MySQL to manage customer purchases and reward points. The system records customer purchases, calculates reward points based on purchase amount slabs, updates customer points automatically, and displays a leaderboard of top customers.

The reward system is based on percentage slabs:

3% reward for purchases up to 2000
5% reward for purchases between 10000
7% reward for purchases above 10000

The system includes customer management, purchase management, reward calculation, leaderboard, and purchase reporting modules. This application helps shop owners manage customer loyalty programs efficiently and maintain accurate purchase and reward records.
